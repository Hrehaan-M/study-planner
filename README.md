PROJECT PROBLEM STATEMENT
Not having enough time to finish work or procrastinating due to not having a set schedule
CONCISE DESIGN OVERVIEW
It will take in all the tasks that need to be completed then it will generate a schedule using greedy bin packing by taking all of the to do list items putting them in an array and organizing them by due date then put them into the schedule in that order based on ho long it will take
SCOPE STATEMENT
The user needs to be able to add and remove tasks from the list and input the due date and hours needed to complete the task and generate the schedule from that
